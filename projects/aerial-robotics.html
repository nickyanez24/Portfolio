Aerial Robotics

Autonomous quadrotor simulation, perception, planning, and control.

Overview

Aerial Robotics was a project-based course focused on autonomous quadrotor systems. The course connected dynamics, controls, state estimation, computer vision, and path planning through simulation and software implementation.

What I Worked On

  • Built and validated a MATLAB quadrotor simulator using 6-DOF rigid body dynamics.
  • Implemented position, altitude, and attitude control algorithms.
  • Simulated GNSS, IMU, and stereo-camera sensor measurements.
  • Implemented computer vision localization using OpenCV and RANSAC.
  • Worked with graph-based path planning algorithms including DFS, Dijkstra, and A*.

Technical Highlights

  • Integrated dynamics, controls, estimation, perception, and planning into an autonomous flight workflow.
  • Used RANSAC to improve localization performance when visual measurements contained false positives.
  • Applied path planning and trajectory generation concepts to a timed autonomous navigation problem.

Tools Used

MATLAB, C++, ROS, OpenCV, RANSAC, A*, Dijkstra, DFS, state estimation, quadrotor dynamics, feedback control
