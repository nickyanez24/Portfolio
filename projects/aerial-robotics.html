Aerial Robotics

Autonomous quadrotor simulation, perception, planning, and control.

Overview

Aerial Robotics was a project-based course focused on autonomous quadrotor systems. Throughout the semester, I worked with the major building blocks required for autonomous flight, including dynamics, control, state estimation, computer vision, path planning, and trajectory generation.

The course connected theory from dynamics and controls with practical implementation in simulation, robotics software, and autonomous navigation challenges.

Project Objectives

  • Develop a quadrotor simulation and control framework.
  • Model vehicle dynamics, aerodynamic drag, and motor behavior.
  • Integrate sensor models and state estimation for noisy measurements.
  • Implement computer vision and localization algorithms.
  • Develop path planning and trajectory generation methods for autonomous navigation.
  • Apply course concepts in a final autonomous competition.

What I Worked On

  • Built and validated a MATLAB quadrotor simulator using 6-DOF rigid body dynamics.
  • Implemented position, altitude, and attitude control algorithms.
  • Added aerodynamic drag and motor dynamics to improve simulation fidelity.
  • Simulated GNSS, IMU, and stereo-camera sensor measurements.
  • Integrated state estimation techniques for tracking under measurement noise.
  • Implemented computer vision localization using OpenCV and RANSAC.
  • Worked with graph-based path planning algorithms including DFS, Dijkstra, and A*.

Final Competition

The course culminated in a team-based final competition with two major challenges. The first challenge required autonomous navigation through a timed obstacle course. The second challenge focused on robust computer vision and required accurate 3D localization of colored balloons in the presence of false detections.

Our team developed an autonomy architecture involving mission planning, global path planning, local trajectory generation, and navigation logic. For the vision challenge, we incorporated a RANSAC-based estimation approach into the balloon localization pipeline to reject outliers and improve localization robustness.

Technical Highlights

  • Connected dynamics, controls, estimation, perception, and planning into a single autonomous system workflow.
  • Used RANSAC to improve localization performance when visual measurements contained false positives.
  • Applied path planning and trajectory generation concepts to a timed autonomous navigation problem.
  • Developed experience debugging robotics software across simulation, perception, and planning modules.

Key Takeaways

  • Autonomous systems depend on the interaction between dynamics, estimation, perception, planning, and control.
  • Simulation assumptions strongly affect controller and estimator behavior.
  • Robust perception is critical when working with noisy or imperfect visual measurements.
  • Testing and debugging are just as important as the initial algorithm design.

Tools Used

MATLAB, C++, ROS, OpenCV, RANSAC, A*, Dijkstra, DFS, state estimation, quadrotor dynamics, feedback control
