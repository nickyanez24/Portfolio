AERODYNAMICS

Vortex Generator Optimization

Wind tunnel testing and flow visualization for drag reduction

This project investigated how vortex generator placement affects aerodynamic drag on a NASCAR-style vehicle. Multiple configurations were tested in a wind tunnel using both flow visualization and quantitative drag measurements.

What I Worked On

  • Researched vortex generator applications from aerodynamic literature.
  • Selected and adapted vortex generator geometries for testing.
  • Conducted wind tunnel testing across multiple Reynolds numbers.
  • Performed flow visualization using smoke and surface tufts.
  • Measured drag forces and calculated drag coefficients.
  • Compared multiple placement configurations against a baseline vehicle.

Technical Highlights

  • Combined qualitative flow visualization with quantitative drag measurements.
  • Analyzed boundary layer behavior and flow separation.
  • Demonstrated that proper vortex generator placement reduced drag by approximately 3%.
  • Showed that poor placement could increase drag despite using the same device.

Tools Used

Wind tunnel testing, flow visualization, aerodynamics, drag analysis, Reynolds number scaling, experimental testing
